Object files are X-Plane .obj files that conform to the specifications found at http://scenery.x-plane.com/library.php?doc=obj8spec.php

No objects should exist in this path. Place objects in appropriate child paths.

Object naming convention.

1. Avoid spaces in file names
2. Use camel case filenames (ie: Modern_Control_Tower.obj)
3. Use sizing in the filename where practical. i.e.:
	Large_Hangar.obj
	Small_Hangar.obj
	GA_Hangar_130x150x80f.obj   (width x length x height, f=feet, m=metres)
